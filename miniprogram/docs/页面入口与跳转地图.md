# 页面入口与跳转地图

> 目标：快速看清“用户从哪里进入、在何处完成任务、最终回到哪里”。

## 1. 顶层导航（Tab）

- 首页：`pages/home/index`
- 趋势：`pages/progress/index`
- 我的：`pages/profile/index`

## 2. 首页主链路

- 新用户首登首页 -> `pages/onboarding-profile/index`
- 首页右下角加号（饮食记录） -> `pages/food-search/index?recordDate=...&mealType=...&source=home`
- 首页右下角加号（运动记录） -> `pages/exercise-search/index?recordDate=...&mode=create&source=home`
- 首页记录流（饮食项） -> `pages/food-search/index?mode=edit&recordId=...&recordDate=...&mealType=...&source=home`
- 首页记录流（运动项） -> `pages/exercise-search/index?mode=edit&recordId=...&recordDate=...&source=home`
- 首页健康日记区 -> `pages/health-diary-editor/index?recordDate=...`

## 3. 饮食链路（当前实现）

- `food-search` 页面内完成“选择 + 编辑”：
  - 搜索关键字时，隐藏左侧分类并在全量食物中检索
  - 支持“最近记录 / 最近搜索 / 自定义 / 套餐 / 分类”筛选
  - 食物列表展示：食物名 + 热量（kcal）
- 选择食物后，打开同页底部弹窗编辑（不再跳独立编辑页）：
  - 新建按钮：`添加`
  - 编辑按钮：`完成编辑`
  - 头部操作：删除图标（编辑态） + 关闭图标
- 自定义食物入口：仅在“自定义”筛选标题右侧显示 `添加` 按钮
- 自定义食物管理页：`profile -> custom-food`
- 选择页创建回流：`food-search -> custom-food -> food-search`

## 4. 运动链路

- `exercise-search` 页面内完成“选择 + 编辑”：
  - 支持“最近运动 / 最近搜索 / 自定义 / 分类”筛选
  - 选择运动后打开同页底部弹窗编辑时长与强度
  - 首页新增运动关闭弹窗后留在 `exercise-search`，编辑已有记录关闭后回首页
- 首页新增：`home -> exercise-search(mode=create)`
- 首页编辑：`home -> exercise-search(mode=edit, recordId=...)`
- 自定义运动管理页：`profile -> custom-exercise`
- 选择页创建回流：`exercise-search -> custom-exercise -> exercise-search`

## 5. 我的页链路

- `profile` -> `personal-info`
- `profile` -> `health-profile`
- `profile` -> `custom-food`
- `profile` -> `meal-combo-manage`
- `profile` -> `custom-exercise`

## 6. 趋势与体征链路

- `health-profile` -> `progress`（查看趋势入口）
- `progress` Tab 内完成“趋势浏览 + 快捷记录 + 历史查看”：
  - 顶部区间切换：`MONTH / YEAR / ALL`
  - 趋势图固定单屏展示，不左右滑动
  - 顶部主数值与日期跟随当前选中点变化
  - `YEAR / ALL` 当前展示日级点；`ALL` 进入后自动拉全量分页结果
  - `YEAR / ALL` 横轴只显示首尾 `YYYY-MM`，纵轴三条水平线显示整数刻度
  - 指标卡右上角 `+`：打开同页体征记录弹窗（BMI 不支持手工新增）
  - 指标卡右上角 `···`：进入 `metric-history`
- 历史页：`progress -> metric-history(metricKey, metricLabel, unit)`
- 历史页支持：
  - 查看单项指标历史明细
  - 对可录入指标再次新增记录

## 7. 套餐管理链路

- `meal-combo-manage` 列表 -> 编辑态（同页）
- 编辑态 -> `food-search`（添加食物）
- 编辑态 -> 查看套餐总热量与三大营养素汇总
- 编辑态 -> 保存套餐更新
- 列表态 -> 删除套餐
